Специальные биты доступа к объектам файловой системы

1. Бит владельца файла (userbit) - позволяет запускать исполняемую программу от имени владельца файла
2. Установка u-бита:
sudo chmod u+s <имя исполняемого файла>
sudo chmod u+s /bin/passwd
sudo chmod u+s script.bash
3. Снятие u-бита:
sudo chmod u-s <имя исполняемого файла>
sudo chmod u-s /bin/passwd
sudo chmod u-s script.bash
4. Просмотр прав доступа к файлу с установленным s-битом:
stat /bin/passwd

s-бит устанавливается только для регулярных файлов
